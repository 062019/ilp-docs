# Vaksinasi/Imunisasi untuk Anak Sekolah
Khususnya untuk jenjang SD/MI sederajat meliputi 
1. MR (Measles Rubela) untuk mencegah Campak dan Rubela, 
2. Td dan Dt untuk mencegah Tetanus dan Difteri, 
3. HPV (Human Papilloma Virus) untuk mencegah Kanker Leher Rahim.

Waktu pemberian disesuaikan dengan tabel 18 untuk Jadwal Bulanan Imunisasi Anak SD/MI

Untuk pemberian vaksin seperti COVID-19 dapat diberikan oleh nakes Puskesmas di luar sekolah. Saat ini vaksin COVID-19 diperbolehkan bagi anak 6-11 tahun dan 12-17 tahun.

Setiap remaja putri diharapkan mencapai **status imunisasi T5** dengan tujuan untuk memberikan perlindungan terhadap penyakit tetanus. Pemberian imunisasi yang mengandung Tetanus toxoid ditentukan berdasarkan hasil skrining status imunisasi T.
