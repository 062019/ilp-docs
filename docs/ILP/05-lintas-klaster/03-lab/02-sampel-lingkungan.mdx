# Pemeriksaan sampel lingkungan
Pengujian kualitas air minum untuk parameter wajib dengan sanitarian kit;

Pengujian kualitas udara untuk parameter fisik dan mikroba dengan sanitarian kit;

Pengujian keamanan pangan untuk parameter E coli dan parameter kimia dengan sanitarian kit;

Identifikasi vektor dan binatang pembawa penyakit secara makroskopis;

Analisis kepadatan vektor dan binatang pembawa penyakit dengan entomologi kit;

Pengambilan sampel untuk rujukan pengujian.
