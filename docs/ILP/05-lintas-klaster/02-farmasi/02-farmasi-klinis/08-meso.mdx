# Monitoring Efek Samping Obat (MESO)

MESO bertujuan untuk menemukan Efek Samping Obat (ESO) sedini mungkin terutama yang berat, tidak dikenal dan frekuensinya jarang.

Pelaporan efek samping obat dapat dilakukan melalui [e-Meso](https://e-meso.pom.go.id).
