# Pelayanan Kesehatan Peduli Remaja (PKPR) di Luar Gedung (Fasilitasi Usaha Kesehatan Sekolah (UKS)/Madrasah)
UKS bertujuan untuk menangani permasalahan kesehatan antara lain masalah gizi termasuk anemia, permasalahan gigi mulut termasuk karies, gangguan indera penglihatan dan pendengaran serta perilaku berisiko pada remaja. Lingkup UKS terdiri dari 
1. pelayanan kesehatan anak sekolah, 
2. pendidikan kesehatan dan 
3. pembinaan lingkungan sekolah sehat.
