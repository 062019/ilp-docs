## Ge­ri­atri
**SDM Kesehatan**: Dok­ter/Pe­ra­wat/Bi­dan
## Pe­ri­laku Me­ro­kok
**SDM Kesehatan**: Dok­ter/ Pe­ra­wat/Bi­dan
## Ting­kat ak­ti­vi­tas fi­sik
**SDM Kesehatan**: Dok­ter/ Pe­ra­wat/Bi­dan
## Sta­tus Gizi (Obe­si­tas, Gizi Ku­rang)
**SDM Kesehatan**: Dok­ter/Pe­ra­wat/Bi­dan/Nut­risi onis

**Alat Kesehatan**: Set An­tro­po­metri
## Gigi
**SDM Kesehatan**: Dok­ter Gigi/ Te­ra­pis Gigi

**Alat Kesehatan**: Set Pe­me­rik­saan Gigi
## Te­kanan da­rah
**SDM Kesehatan**: Dok­ter/Pe­ra­wat/Bi­dan

**Alat Kesehatan**: Ten­si­me­ter
## Gula darah
**SDM Kesehatan**: Perawat/Bidan/TTLM

**Alat Kesehatan**: Glukometer

**Bahan Medis Habis Pakai (BMHP)**: 
- Strip gula darah
- alkohol swab, Lancet
:::tip Perhitungan
```math
BMHP=Sasaran
```
:::
- $BMHP$ $=$ Jumlah kebutuhan BMHP Gula darah
- $Sasaran$ $=$ Jumlah sasaran lanjut usia (>60 tahun)

## Risiko Stroke
**SDM Kesehatan**: TTLM

**Alat Kesehatan**: Clinical Chemistry Analyzer/Fotometer

**Bahan Medis Habis Pakai (BMHP)**
- Reagen total kolesterol
- Reagen Trigliserid,
- Reagen HDL
- Jarum vacutainer,
- Tabung vacutainer,
- Alkohol swab,
- Plester
- Tip kuning, Bahan kontrol
## Risiko Jantung
**SDM Kesehatan**: Dokter/Perawat
## EKG
**Bahan Medis Habis Pakai (BMHP)** 
- Thermal paper EKG
- Gel EKG
## Fungsi ginjal
**SDM Kesehatan**: TTLM

**Alat Kesehatan**: Fotometer / Chemistry Analyzer

**Bahan Medis Habis Pakai (BMHP)** 
- Pemeriksaan serum Ureum dan Kreatinin dengan BMHP
- Reagen ureum
- Reagen kreatinin
- Yellow Micropipette Tips
- Hand Scoon
- Jarum Vacutainer 22
- Alcohol Swab
- Plester
- Tabung Vacutainer Tutup Kuning
:::tip Perhitungan
```math
BMHP=(DM+HT)\ x\ Sasaran
```
:::
- $BMHP$ $=$ Jumlah kebutuhan BMHP Risiko stroke, risiko jantung, dan fungsi ginjal 
- $DM$ $=$ Prevalensi DM
- $HT$ $=$ Prevalensi HT
- $Sasaran$ $=$ Jumlah sasaran lanjut usia (>60 tahun)

## Tuberkulosis
**SDM Kesehatan**: Dokter/Bidan/ Perawat

**Alat Kesehatan**: Mikroskop atau mesin TCM

**Bahan Medis Habis Pakai** (BMHP): Tes Sputum BTA / TCM
:::tip Perhitungan
```math
BMHP=TB\ x\ Sasaran
```
:::
- $TB$ $=$ Prevalensi Tuberkulosis 
- $Sasaran$ $=$ jumlah sasaran lanjut usia (>60 tahun)

## Penyakit Paru Obstruktif Kronis
**SDM Kesehatan**: Dokter/ Perawat/Bidan
## Kanker Payudara
**SDM Kesehatan**: Dokter/Perawat/Bidan

**Alat Kesehatan**: USG dengan Probe Linier

**Bahan Medis Habis Pakai** (BMHP): Gel pelicin/pelumas untuk pasien/lubricant gel untuk USG
## Kanker Leher Rahim
**SDM Kesehatan**: Dokter/Bidan

**Alat Kesehatan**: PCR[^*]

**Bahan Medis Habis Pakai** (BMHP): 
- Collecting Kit (cytobrush + VTM),
- Reagen Ekstraksi,
- Reagen PCR
:::tip rumus
```math
BMHP=Sasaran
```
:::
$Sasaran$ $=$ jumlah sasaran perempuan lanjut usia (>60 tahun)

## Kanker Paru
**SDM Kesehatan**: Dokter/ Perawat/Bidan
## Kanker Usus
**SDM Kesehatan**: Dokter/ Perawat/Bidan

**Bahan Medis Habis Pakai** (BMHP): 
- Pot feses
- sarung tangan non steril
- FOBT/Reagen darah samar
:::tip rumus
```math
BMHP=Sasaran
```
:::
$Sasaran$ $=$ Jumlah sasaran laki laki lanjut usia (> 60 tahun)
## Mata
**SDM Kesehatan**: Dokter/ Bidan/ Perawat

**Alat Kesehatan**: 
- Oftalmoskop
- Senter
- _E-Tumbling / Snellen Chart /E- Chart_
- _pinhole_
## Telinga
**SDM Kesehatan**: Dokter/ Bidan/ Perawat

**Alat Kesehatan**: 
- Garpu tala 512 Hz
- otoskop
## Kesehatan Jiwa
**SDM Kesehatan**: Dokter/ Perawat/ Bidan/Psikolog Klinis
## Hepatitis B
**SDM Kesehatan**: TTLM

**Bahan Medis Habis Pakai** (BMHP): 
- Rapid test Hepatitis B (HbSAg),
- Blood lancet,
- Alkohol swab
:::tip rumus
```math
BMHP=HepB\ x\ Sasaran
```
:::
- $HepB$ $=$ Prevalensi Hepatitis B 
- $Sasaran$ $=$ Jumlah sasaran lanjut usia (> 60 tahun)

## Hepatitis C
**SDM Kesehatan**: TTLM

**Bahan Medis Habis Pakai** (BMHP): 
- Rapid test Hepatitis C (anti HCV),
- Blood lancet,
- Alkohol swab
:::tip rumus
```math
BMHP=HepC\ x\ Sasaran
```
:::
- $HepC$ $=$ Prevalensi Hepatitis C 
- $Sasaran$ $=$ Jumlah sasaran lanjut usia (> 60 tahun)


## Fibrosis/ Sirosis hati
**SDM Kesehatan**: TTLM

**Alat Kesehatan**: 
- _Hematology Analyzer,_
- _Chemistry Analyzer_ /Fotometer

**Bahan Medis Habis Pakai (BMHP)**
- BMHP pemeriksaan SGOT
- BMHP pemeriksaan jumlah trombosit
:::tip Perhitungan
```math
Kebutuhan= (HepB+HepC+Ob+Lip)\ x\ Sasaran
```
:::
- $Kebutuhan$ $=$ Kebutuhan BMHP Pemeriksaan Fibrosis/Sirosis Hati
- $HepB$ $=$ Prevalensi Hepatitis B
- $HepC$ $=$ Prevalensi Hepatitis C
- $Ob$ $=$ Prevalensi Obesitas Sentral
- $Lip$ $=$ Prevalensi Dislipidemia
- $Sasaran$ $=$ Jumlah sasaran lanjut usia (> 60 tahun)

[^*]: Di Labkesmas Tingkat 2

Sebelum melakukan perhitungan BMHP dilakukan penghitungan jumlah sasaran. Jumlah sasaran Lanjut Usia adalah jumlah penduduk usia >60 tahun.

:::tip Rumus
```math
Estimasi = jumlah + 10\%
```

:::
- $Estimasi = $ kebutuhan BMHP setahun 
- $jumlah = $ sasaran dalam setahun 
- $10\% = $ kemungkinan perpindahan atau penambahan sasaran

**Catatan:**

Untuk setiap jenis BMHP fungsi ginjal (misalnya, _yellow micropipette tips_, _cuvet_ 4,5 mL, tabung vacutainer tutup kuning, dan lain-lain), rumus yang digunakan adalah:
:::note Rumus BMHP Fungsi Ginjal 
```math
BMHP = Sasaran\ x\ Pemakaian
```
<br/>
:::
- $BMHP =$ Kebutuhan BMHP 
- $Sasaran =$ Sasaran Skrining 
- $Pemakaian =$ Pemakaian BMHP per Individu

Secara umum, perhitungan BMHP harus memperhitungkan stok sisa tahun sebelumnya, sehingga penghitungan menjadi:
:::tip Sisa Stok
```math
Akhir = Kebutuhan + Buffer - Sisa 
```
<br/>
:::
- $Akhir =$ Kebutuhan Akhir
- $Sisa =$ Sisa Stok Tahun Sebelumnya

Untuk menjamin tidak terjadi kekosongan BMHP, maka Puskesmas, FKTP lain, FKTL dan fasilitas lainnya yang ditunjuk harus selalu memiliki stok BMHP yang mencukupi untuk pelaksanaan PKG selama 3 (tiga) bulan ke depan.

Puskesmas, FKTP lain, dan fasilitas penyelenggara PKG lainnya dapat meminta tambahan stok BMHP ke Dinas Kesehatan Kabupaten/Kota atau Puskesmas sesuai tata hubungan kerja yang ditentukan, setiap bulan sesuai dengan jumlah BMHP yang sudah digunakan pada bulan sebelumnya.
